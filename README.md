## 動かし方
このプロジェクトはuvを使って管理しており、それを使って動かすことを想定している。

### uv
1. `main.py`と同じディレクトリに加工したい画像を`sample_image.jpg`という名前で保存。
2. シェル上で次のコマンドを実行
```
uv run main.dy
```
3. ディレクトリ内に加工後の画像が`edgetoon_image.jpg`という名前で保存されている。
